# Code de conduite — Opays Intelligence Platform

## Notre engagement

Dans l’interet de favoriser un environnement ouvert et accueillant, nous, contributeurs et mainteneurs, nous engageons a faire de la participation a OIP une experience exempte de harcelement pour tout le monde.

## Nos standards

Exemples de comportements contribuant a un environnement positif :

- Utiliser un langage accueillant et inclusif
- Respecter les points de vue et les experiences differents
- Accepter la critique constructive avec calme
- Orienter la discussion vers l’amelioration du produit
- Montrer de l’empathie envers les autres contributeurs

Exemples de comportements inacceptables :

- Usage de langage ou d’images a caractere sexuel ou discriminatoire
- Troll, commentaires insultants ou personnels
- Harcelement public ou prive
- Divulgation d’informations privees sans autorisation
- Tout autre comportement pouvant etre considere comme inapproprie dans un cadre professionnel

## Responsabilite

Les mainteneurs du projet ont le droit et la responsabilite de supprimer, modifier ou rejeter les contributions qui ne respectent pas ce code de conduite.  
Ils peuvent aussi bannir temporairement ou definitivement un contributeur en cas de comportement inapproprie.

## Portee

Ce code de conduite s’applique a tous les espaces du projet : depot, issues, pull requests, discussions, et espaces publics ou l’individu represente le projet.

## Signalement

Les abus peuvent etre signales a l’equipe OIP via les canaux internes d’Opays.

## Licence
